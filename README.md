# 🚚 Supply Chain Analytics Dashboard

![Supply Chain Banner](https://images.unsplash.com/photo-1586528116311-ad8dd3c8310d?w=1200)

## 📌 Project Overview

This project focuses on analyzing supply chain operations to identify trends, improve operational efficiency, and support data-driven decision-making.

The analysis was performed using Python for data cleaning, exploration, and visualization, and the final dashboard was designed to present key business insights in an interactive format.

---

## 🎯 Business Objectives

- Monitor supply chain performance
- Analyze sales and profit trends
- Identify high-performing products and categories
- Track customer and regional performance
- Improve operational efficiency through data insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- HTML Dashboard
